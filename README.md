# M14-Tech-Blog

## Description
 This app follows the MVC paradigm in its architectural structure, using Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication. This app allows the user to publish their blog posts and comment on other developersâ€™ posts as well.
